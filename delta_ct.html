Delta CT Method

Take an input table with Ct values, sample labels for group, timepoint, and target gene, and calculate relative expression as Delta Ct or Delta-Delta Ct values.

Input file

Input should be a single Excel file with at least the following columns:

  • Sample: Unique identifier for each sample.
  • Group: Experimental group (e.g., Control, Treatment).
  • Timepoint: Timepoint of the sample (e.g., 0h, 24h).
  • Target: Target gene (or primer pair) name.
  • Ct: Ct value for the target gene.

Parameters

  • Reference Target: The target gene used for normalization (e.g., GAPDH).
  • Reference Timepoint: The timepoint used as a baseline for Delta-Delta Ct calculations (e.g., 0h).
